# Город в объективе

Сайт-портфолио фотографа, специализирующегося на городской съёмке.

## О проекте

Проект представляет собой полноценный веб-сайт для демонстрации портфолио и приёма заявок от клиентов. Сайт разработан для трёх типов пользователей: потенциальных клиентов (20–40 лет), моделей/стилистов для коллабораций, а также друзей и партнёров фотографа.

**Основные функции:**
- Галерея портфолио с фильтрацией по категориям
- Полноэкранный просмотр фотографий
- Форма обратной связи с сохранением данных в БД
- Адаптивное меню и удобная навигация

## Установка и развёртывание

## 📖 Руководство по установке и настройке форума (bbPress)

Данное руководство описывает шаги по добавлению функционала форума на сайт-портфолио фотографа.

### 📋 Требования
- Установленный WordPress (версия 5.0+).
- Хостинг с поддержкой PHP 7.4+ и MySQL.
- Доступ в админ-панель WordPress.

### 🛠 Процесс установки

#### Шаг 1: Установка плагина
1. В админ-панели перейдите в раздел **Плагины → Добавить новый**.
2. В строке поиска введите `bbPress`.
3. Найдите плагин от разработчиков bbPress и нажмите **"Установить"**, затем **"Активировать"**.
АктивныеПлагины.png

#### Шаг 2: Создание страниц форума
1. После активации перейдите в **Форумы**.
2. Создайте несколько форумов и категорий, настраивая на правой панели их взаимное расположение.
3. После создания перейдите в **Страницы** и создайте новую, добавив форму форума.
ФормыФорумаВАдминке.png

#### Шаг 3: Настройка внешнего вида (Стилизация)
1. Установите плагин **"bbp style pack"** (рекомендуется для кастомизации).
2. Перейдите в **Настройки → bbp style pack**.
3. Во вкладке **"Topic/Reply Form"** активируйте пункт 10 и добавьте текст **"Правила сообщества"**.
