# Город в объективе

Сайт-портфолио фотографа, специализирующегося на городской съёмке.

## О проекте

Проект представляет собой полноценный веб-сайт для демонстрации портфолио и приёма заявок от клиентов. Сайт разработан для трёх типов пользователей: потенциальных клиентов (20–40 лет), моделей/стилистов для коллабораций, а также друзей и партнёров фотографа.

**Основные функции:**
- Галерея портфолио с фильтрацией по категориям
- Полноэкранный просмотр фотографий
- Форма обратной связи с сохранением данных в БД
- Адаптивное меню и удобная навигация

## Технологический стек

| Компонент | Технология |
|-----------|-------------|
| Операционная система | Linux (Ubuntu / CentOS) |
| Веб-сервер | Apache |
| Язык программирования | PHP |

| CMS | WordPress |
| Система контроля версий | Git + GitHub |

## Структура сайта

| Страница | Содержание |
|----------|------------|
| Главная | Приветственное фото, имя фотографа, слоган «Снимаю ритм большого города», кнопка перехода в портфолио |
| Портфолио | Галерея работ с фильтрацией по трём категориям: «Люди в городе», «Архитектура», «Репортаж» |
| Услуги и цены | Список видов съёмок (часовая, тематическая) с базовыми ценами или пометкой «под запрос» |
| О фотографе | Рассказ о стиле, опыте, используемой технике |
| Контакты | Телефон, Telegram, Email и форма обратной связи |
| Советы | FAQ: подготовка к съёмке, действия при дожде, сроки получения фото |
