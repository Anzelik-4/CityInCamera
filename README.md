# Город в объективе

Сайт-портфолио фотографа, специализирующегося на городской съёмке.

## О проекте

Проект представляет собой полноценный веб-сайт для демонстрации портфолио и приёма заявок от клиентов. Сайт разработан для трёх типов пользователей: потенциальных клиентов (20–40 лет), моделей/стилистов для коллабораций, а также друзей и партнёров фотографа.

**Основные функции:**
- Галерея портфолио с фильтрацией по категориям
- Полноэкранный просмотр фотографий
- Форма обратной связи с сохранением данных в БД
- Адаптивное меню и удобная навигация

## Установка и развёртывание

## 📖 Руководство по установке и настройке форума (bbPress)

Данное руководство описывает шаги по добавлению функционала форума на сайт-портфолио фотографа.

### 📋 Требования
- Установленный WordPress (версия 5.0+).
- Хостинг с поддержкой PHP 7.4+ и MySQL.
- Доступ в админ-панель WordPress.

### 🛠 Процесс установки

#### Шаг 1: Установка плагина
1. В админ-панели перейдите в раздел **Плагины → Добавить новый**.
2. В строке поиска введите `bbPress`.
3. Найдите плагин от разработчиков bbPress и нажмите **"Установить"**, затем **"Активировать"**.
АктивныеПлагины.png

#### Шаг 2: Создание страниц форума
1. После активации перейдите в **Форумы**.
2. Создайте несколько форумов и категорий, настраивая на правой панели их взаимное расположение.
3. После создания перейдите в **Страницы** и создайте новую, добавив форму форума.
ФормыФорумаВАдминке.png

#### Шаг 3: Настройка внешнего вида (Стилизация)
1. Установите плагин **"bbp style pack"** (рекомендуется для кастомизации).
2. Перейдите в **Настройки → bbp style pack**.
3. Во вкладке **"Topic/Reply Form"** активируйте пункт 10 и добавьте текст **"Правила сообщества"**.
ПравилаОтветов.png

## ✅ Чек-лист тестирования функционала форума
Перед сдачей проекта необходимо убедиться, что следующие сценарии работают ожидаемо.
### 🧩 Общая функциональность
- [ ] **Отображение страниц**: Все созданные страницы форума имеют стандартный дизайн.
- [ ] **Адаптивность**: Форум корректно отображается на мобильных устройствах.
- [ ] **Русский язык**: Названия кнопок ("Reply", "Create Topic") переведены на русский.
### 👤 Роли и права доступа
- [ ] **Гость (Неавторизованный)**: Видит список тем, но при попытке написать ответ видит сообщение "Вы должны войти в систему".
- [ ] **Участник**: Может создавать новые темы и отвечать в существующих.
- [ ] **Модератор/Админ**: Видит кнопки "Изменить" и "Удалить" под сообщениями других пользователей.
### ✍️ Создание контента
- [ ] **Создание темы**: Переход по кнопке "Создать тему" открывает редактор, тема сохраняется в БД.
- [ ] **Визуальный редактор**: Работают базовые кнопки форматирования текста (жирный, курсив, список).
- [ ] **Правила**: Над формой создания новой темы отображается текст "Правила сообщества", который добавлен через `bbp style pack`.
### 📊 Администрирование (Back-end)
- [ ] **Управление разделами**: Администратор может редактировать описание существующих разделов.
- [ ] **Управление пользователями**: В разделе "Пользователи" администратор видит выпадающий список "Роль на форуме" (Keymaster, Moderator, Participant, Spectator, Blocked).
- [ ] **Управление темами**: Администратор может массово удалять темы ("Корзина").
### 🔗 Интеграция с сайтом
- [ ] **Ссылка в меню**: В главном меню сайта появился пункт **"Форум"**, ведущий на страницу со списком разделов.
- [ ] **Профиль пользователя**: При клике на аватар пользователя в посте форума открывается его стандартная страница профиля WordPress (или страница профиля bbPress).
